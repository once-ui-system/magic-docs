Magic Docs is a modern, MDX-based documentation system built with Once UI, providing a seamless and intuitive way to create, organize, and present technical documentation. Designed with both developers and end-users in mind, Magic Docs combines the flexibility of Markdown with the power of React components to deliver a rich documentation experience.

## Key Features

Magic Docs offers a comprehensive set of features that make documentation creation and consumption a breeze:

- **MDX Integration**: Write content in Markdown and seamlessly embed React components
- **Hierarchical Navigation**: Intuitive section-based navigation with custom ordering
- **Component Showcase**: Interactive examples of Once UI components
- **Responsive Design**: Perfect viewing experience across all devices
- **Dark/Light Mode**: Automatic theme switching based on system preferences
- **Search Functionality**: Powerful command palette for quick content access
- **SEO Optimization**: Built-in metadata generation for better search engine visibility
- **Custom Ordering**: Control the exact order of pages using meta.json files

## Built with Once UI

Magic Docs leverages the Once UI component system, providing a consistent, accessible, and beautiful interface. The documentation itself serves as both a guide and a showcase of what's possible with Once UI components.
